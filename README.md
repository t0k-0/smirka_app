# Šmírka Mobile

### Popis
Šmírka Mobile (GliderLog) je software vytvořen pomocí **Claude AI (Sonnet 4.6)** jako *HTML* soubor. Má za úkol usnadnit zápis startů kluzáků a jejich posádek během letového dne. Disponuje funkcí exportu logu do dokumentu PDF.
### Funkce
Aplikace je přizpůsobena pro použití na obrazovce mobilního telefonu. Následuje seznam funkcí ve smyslu zleva doprava, shora dolů.
##### *Seznam funkcí*

| Funkce                      | Popis                                                                                                                                                                                                                                                                                                                                                                  |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
|                             | LOKACE: ***HORNÍ LIŠTA***                                                                                                                                                                                                                                                                                                                                              |
| Tlačítko "LOG"              | *Stáhne horní lištu s celým logem letového dne. Lety jsou spárovány (kluzák - vlečná) a řazeny chronologicky. Aplikace umožňuje filtrovat (dle pilota, letadla) pro usnadnění souhrnného zápisu za letový den. Vlevo nahoře se nachází tlačítko "EXPORT PDF" umožňující stažení / tisk logu ve formátu PDF (stále testujeme)*                                          |
| Datum                       | *Zobrazuje datum dle zařízení. Toto datum se propíše do PDF souboru za letový den v případě exportu.*                                                                                                                                                                                                                                                                  |
| Tlačátko "Dark / Light"     | *Přepíná mezi světlým a tmavým vzhledem aplikace.*                                                                                                                                                                                                                                                                                                                     |
| Tlačítko "+ Presets"        | *Umožňuje uživateli zadat presety pilotů a letadel pro letový den. U letadla jsou sloty pro registrační značku (imatrikulaci), typ letadla, roli letadla pro letový den (vlečná - TOW, kluzák - GLIDER, motorové a samostart - MOTOR, počet míst. V případě, že je role letadla TOW, nemůže mít více než jednoho pilota. Při zadávání pilota stačí zapsat jeho jméno.* |
|                             | LOKACE: ***HLAVNÍ OBLAST OBRAZOVKY***                                                                                                                                                                                                                                                                                                                                  |
| Horní panel                 | *Zobrazuje probíhající lety a jejich počet. Při kliknutí na let umožňuje zapsat čas přistání letadla.*                                                                                                                                                                                                                                                                 |
| Přepínač "AEROTOW / SINGLE" | *Přepíná mezi zadáváním aerovleků (2 sloty) a samostatných vzletů (1 slot).*                                                                                                                                                                                                                                                                                           |
| Letadlové sloty             | *Umožňují sestavení vzletové konfigurace pomocí přetažení pilotů a letadel na jejich místa.*                                                                                                                                                                                                                                                                           |
| Spodní panel                | *Ukazuje proměnlivé informace v závislosti na zadávání informací do slotů.*                                                                                                                                                                                                                                                                                            |
